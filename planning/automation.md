# Automation
## Infrastruction
- Terraform for provisioning TrueNAS system datasets (pictures, videos, music, documents)
## Configuration
- Ansible for configuring TrueNAS system
    - Install plugins (Nextcloud, Plex, etc.)
    - Configure network settings
    - Set up user accounts and permissions
    - Schedule regular backups
    - Monitor system health and performance
## CI/CD` 
- Jenkins for automating the deployment of updates and patches to TrueNAS system
    - Build and test updates in a staging environment
    - Deploy updates to production environment
    - Rollback updates if necessary
